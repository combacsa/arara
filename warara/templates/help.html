모든 페이지에 적용

단축키

  • 모든 단축키는 Caps Lock이 off된 상태를 기준으로 합니다. Caps Lock on상태에서 오작동할 수 있습니다.
  • t : 커서가 게시판 목록으로 이동한다. 다시 누르면 커서가 컨텐츠로 이동한다.
  • y : 커서가 탑 메뉴로 이동한다. 다시 누르면 커서가 컨텐츠로 이동한다.
  • x : 커서가 컨텐츠로 이동한다. t, y와 다르게 토글이 아님
  • j, k : 커서가 게시판 목록이나 탑 메뉴에 있을때 상하 또는 좌우로 이동한다.
  • space : 커서가 있는 곳의 링크로 연결된다. (단, 브라우저 자체 커서로 인해 엉뚱한 결과가 발생할 수 있음)
  • ' : space와 같은 기능.
  • e : ksearch에 포커스가 생긴다.
  • b : 커서가 유저팝업 앵커를 포함하고 있을때 유저팝업창을 띄운다.
  • f : 메세지보내기 팝업창이 떠있을때 내용 쓰는 textarea로 포커스.
  • B : 유저정보 팝업창이 떠있을 때 블랙리스트에 추가한다.
  • , . : 위아래로 스크롤
  • esc : 단축키가 먹히지 않을때 누르면 50% 확률로 단축키가 먹히게 만들 수 있다.

인덱스 페이지

단축키

  • q : logout

계정 일반

단축키

  • x : 커서가 컨텐츠에 생성된다.

내 계정

  • ID, nickname, signature, introduction, language : 가장 최근에 업데이트한 정보 기록
  • Last login IP: 가장 최근에 접속한 IP주소
  • Last logout time: 가장 최근에 로그아웃한 시각

정보 수정

  • 초기에 nickname에 focus가 있다.
  • Nickname은 register때와 마찬가지로 중복되거나 빈칸일 경우 거부된다.

비밀번호 수정

  • 현재 비밀번호를 다시 입력하거나, 새 비밀번호와 재입력된 새 비밀번호가 일치하지 않을 경우 거부된다.

계정 삭제

  • 계정 삭제 이후에는 자신이 쓴 글을 삭제할 수 없으며, 계정 삭제 전에 직접 삭제하여야 한다.

글목록 페이지

단축키

  • s : 검색방법 컨트롤로 커서 이동. 이미 커서가 있을시 검색창에 포커스가 생긴다.
  • space, ' : 컴색방법에 커서가 있을 때 검색방법 on/off 바꿈.
  • q : index페이지로 이동
  • R : 글쓰기 페이지로 이동

글읽기 페이지

단축키

  • shift + enter : reply쓰기 본문 textarea에 커서가 있을때 글쓰기를 완료한다

글쓰기 페이지

단축키

  • shift + enter : 글쓰기 본문 textarea에 커서가 있을때 글쓰기를 완료한다

쪽지 페이지 일반

단축키

  • u : inbox로 이동
  • o : outbox로 이동
  • s : send로 이동

쪽지 목록 페이지

일반

  • 쪽지 전체체크 : 열제목 행에 있는 체크박스를 클릭하면 화면에 보이는 모든 쪽지가 체크됩니다. 이 때 삭제버튼을 누르거나 d키를 누르면 현재 페이지의 모든 쪽지를 삭제할 수 있습니다. 자바스크립트가 작동하지 않는 사용자는 열제목 행에 있는 체크박스를 눌러도 전체 쪽지가 체크되지 않는데, 이때는 그냥 삭제버튼을 누르면 현재 페이지의 모든 쪽지가 삭제됩니다.

단축키

  • q : index페이지로 이동
  • j, k : 커서가 쪽지 목록 위에 있을 때 커서를 위아래로 이동. 커서가 맨위나 맨아래 있을때는 자동으로 이전/다음페이지로 이동.
  • n, p : 페이지 뒤로, 앞으로 넘김
  • N, P : 페이지그룹 뒤로, 앞으로 넘김
  • J, K : 커서가 쪽지 목록 위에 있을 때 커서를 위아래로 드래그
  • c : 커서가 있는 부분 혹은 드래그된 부분의 체크박스를 토글한다.
  • d : 체크된 메세지 삭제. 체크된 메세지가 없으면 커서가 있는 부분 혹은 드래그된 부분의 메세지를 삭제한다.
  • A : 쪽지 전체 선택
  • space : 커서가 있는 부분의 메세지를 읽는다.

쪽지 읽기 페이지

단축키

  • q : 쪽지 목록 페이지로 이동
  • d : 현재 쪽지 삭제
  • (inbox) r, m : 답장보내기에 커서 포커스

단축키

  • shift + enter : 쪽지를 보낸다
