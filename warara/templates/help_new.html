아라 단축키 모음

단축키는 대소문자를 구별합니다.

  • q : main page로 이동한다. main page에서는 로그아웃
  • , : 아래로 스크롤
  • . : 위로 스크롤
  • j+i : 다음글
  • k+i : 이전글

  • j : 글 선택 커서를 위로 이동한다.
  • k : 글 선택 커서를 아래로 이동한다.
  • ', i, space : 선택된 글 보기

  • l : 글 리스트 보기
  • N : 다음 글 페이지
  • P : 이전 글 페이지
  • s : 검색 방법 컨트롤로 이동한다. 이미 커서에 있을 때는 검색창으로 커서가 옮겨간다.
  • R : 글쓰기
  • shift + Enter : 글 쓰는 중 글쓰기를 완료한다.
